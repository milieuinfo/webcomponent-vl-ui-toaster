# vl-ui-toaster
De toaster component is een container voor een aantal(maximame 5) gestapelde alerts. Als de gebruiker een alert sluit, wordt deze uit de toaster verwijderd.

## Installation
```
npm install --save vl-ui-button
```

## Demo
```
npm run demo
```
